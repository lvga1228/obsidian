## Skill Content

# 聚类与分群分析 (Clustering & Segmentation)

## Purpose
使用无监督学习（K-Means）对客户/用户/产品进行分群，通过肘部法则确定最优簇数，输出每个分群的业务画像和可操作建议。

## When to use

- 需要将客户分为不同群体制定差异化策略。
- 探索性发现数据中的自然分组。
- 无标签数据的初步分类。
- 用户说"帮我做个客户分群"、"看看哪些用户相似"。

## Do not use when

- 已有明确标签的分类任务（用分类模型）。
- 数据量极大（>100 万行，K-Means 会慢）— 考虑 MiniBatchKMeans。
- 簇形状非球形 — 考虑 DBSCAN。

## Core Workflow (核心工作流)

### Step 1: 导入与探索
```python
import numpy as np
import pandas as pd
import matplotlib.pyplot as plt
import seaborn as sns
from sklearn.cluster import KMeans
from sklearn.preprocessing import StandardScaler
import plotly.graph_objs as go
import plotly.offline as py

df = pd.read_csv('customers.csv')
print(f"Shape: {df.shape}")
print(df.describe())
```

### Step 4: 确定最优簇数 — 肘部法则 (Elbow Method)
```python
# 计算不同 K 值下的 Inertia（簇内平方和）
inertias = []
K_range = range(1, 11)

for k in K_range:
    kmeans = KMeans(n_clusters=k, init='k-means++', random_state=42, n_init=10)
    kmeans.fit(df[['Age', 'Spending Score (1-100)']])
    inertias.append(kmeans.inertia_)

# 画肘部曲线
plt.figure(figsize=(10, 6))
plt.plot(K_range, inertias, 'bo-')
plt.xlabel('Number of Clusters (K)')
plt.ylabel('Inertia (Within-Cluster Sum of Squares)')
plt.title('Elbow Method for Optimal K')
plt.xticks(K_range)
plt.grid(True, alpha=0.3)
plt.show()
```

### Step 5: 执行 K-Means 聚类
```python
# 选择最优 K（通常肘部拐点）
optimal_k = 4

# 注意：多特征时需标准化
scaler = StandardScaler()
X = df[['Age', 'Annual Income (k$)', 'Spending Score (1-100)']]
X_scaled = scaler.fit_transform(X)

kmeans = KMeans(n_clusters=optimal_k, init='k-means++', random_state=42, n_init=10)
df['Cluster'] = kmeans.fit_predict(X_scaled)

print(f"Cluster sizes:\n{df['Cluster'].value_counts().sort_index()}")
```

### Step 6: 可视化聚类结果
```python
# 2D 散点图（选两个最重要维度）
plt.figure(figsize=(12, 8))
colors = ['red', 'blue', 'green', 'purple', 'orange']
for i in range(optimal_k):
    cluster_data = df[df['Cluster'] == i]
    plt.scatter(
        cluster_data['Annual Income (k$)'],
        cluster_data['Spending Score (1-100)'],
        c=colors[i], label=f'Cluster {i}', alpha=0.7, s=50
    )
    # 标记簇中心
    center = kmeans.cluster_centers_[i]
    # 反标准化中心点
    center_orig = scaler.inverse_transform([center])[0]
    plt.scatter(center_orig[1], center_orig[2], c=colors[i], marker='X', s=200, edgecolors='black')

plt.xlabel('Annual Income')
plt.ylabel('Spending Score')
plt.title('Customer Segments')
plt.legend()
plt.show()
```

### Step 8: 客户画像分析 (Cluster Profiling)
```python
# 每个簇的统计特征
profile = df.groupby('Cluster').agg({
    'Age': ['mean', 'median', 'std'],
    'Annual Income (k$)': ['mean', 'median', 'std'],
    'Spending Score (1-100)': ['mean', 'median', 'std'],
    'Gender': lambda x: (x == 'Male').mean()  # 男性占比
}).round(2)

print(profile)

# 为每个簇命名（业务语义）
cluster_names = {
    0: 'Low Income, Low Spending',
    1: 'Low Income, High Spending',  
    2: 'High Income, Low Spending',
    3: 'High Income, High Spending',
}
df['Segment'] = df['Cluster'].map(cluster_names)
```

### Step 9: 雷达图 (Radar Chart) — 多维度对比
```python
from math import pi

# 簇中心（标准化后）
centers = kmeans.cluster_centers_
feature_names = ['Age', 'Annual Income', 'Spending Score']
n_features = len(feature_names)

angles = [n / float(n_features) * 2 * pi for n in range(n_features)]
angles += angles[:1]  # 闭合

fig, ax = plt.subplots(figsize=(8, 8), subplot_kw=dict(polar=True))
for i in range(optimal_k):
    values = centers[i].tolist()
    values += values[:1]
    ax.plot(angles, values, 'o-', linewidth=2, label=f'Cluster {i}')
    ax.fill(angles, values, alpha=0.1)

ax.set_xticks(angles[:-1])
ax.set_xticklabels(feature_names)
ax.set_title('Cluster Profiles — Radar Chart')
ax.legend(loc='upper right', bbox_to_anchor=(1.2, 1.1))
plt.show()
```

## Quality Checklist (自检表)

- [ ] 数据是否标准化后再聚类（StandardScaler）？
- [ ] 是否用肘部法则或轮廓系数确定了最优 K 值？
- [ ] 是否检查了各簇的样本量是否均衡（避免空簇/极小簇）？
- [ ] 是否对每个簇生成了可读的业务画像？
- [ ] 是否使用了多个 K 值对比结果？
- [ ] 是否检查了特征的多重共线性（避免某些特征主导距离计算）？

## Common Pitfalls

1. **不标准化数据**：距离度量会被量级大的特征主导。
2. **只看一个 K 值**：必须画肘部曲线，对比多个 K。
3. **忽略业务可解释性**：统计最优≠业务最优，选4类可能比选6类更好解释。
4. **用分类变量直接聚类**：K-Means 需要数值变量，分类变量需先编码或使用 K-Prototypes。
5. **不检查异常值**：极端值会严重扭曲簇中心。

## Bonus: 轮廓系数 (Silhouette Score)
```python
from sklearn.metrics import silhouette_score

sil_scores = []
for k in range(2, 11):
    kmeans = KMeans(n_clusters=k, random_state=42, n_init=10)
    labels = kmeans.fit_predict(X_scaled)
    sil_scores.append(silhouette_score(X_scaled, labels))

plt.plot(range(2, 11), sil_scores, 'bo-')
plt.xlabel('K')
plt.ylabel('Silhouette Score')
plt.title('Silhouette Method')
plt.show()
```
