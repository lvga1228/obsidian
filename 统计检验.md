## Skill Content

# 统计检验方法 (Statistical Testing Methods)

## Purpose
提供系统性的统计检验方法论：从描述性统计到推断性统计，覆盖参数检验、非参数检验、相关性分析、效应量计算与正态性检验。为专业数据分析报告提供统计严谨性支撑。

## When to use
- 需要比较两组或多组数据的均值差异
- 需要判断变量间是否存在显著关联
- 需要检验数据是否服从正态分布
- 需要计算效应量来量化差异大小
- 用户说"有没有显著性差异"、"做统计检验"、"p值多少"、"数据正态吗"
- 撰写学术报告或专业数据分析报告时

## 1. 描述性统计 — 数据的第一印象

### 1.1 describe() — 八数概括

```python
import pandas as pd

df = pd.read_csv("data.csv")
df.describe()
# 输出: count, mean, std, min, 25%, 50%, 75%, max
```

**各指标含义：**

| 指标 | 含义 | 解读要点 |
|------|------|----------|
| count | 非缺失值行数 | 与总行数对比判断缺失程度 |
| mean | 算术平均数 | 受极端值影响大 |
| std | 标准差 | 衡量数据离散程度 |
| min | 最小值 | 检查是否有逻辑错误（如负数年龄） |
| 25% | 第25百分位数 / Q1 | IQR = Q3 - Q1 |
| 50% | 第50百分位数 / 中位数 | 比均值更抗异常值 |
| 75% | 第75百分位数 / Q3 | 与Q1共同构成箱体范围 |
| max | 最大值 | 与min配合做范围检查 |

**异常值检测（IQR 方法）：**
```python
Q1 = df['col'].describe()['25%']
Q3 = df['col'].describe()['75%']
IQR = Q3 - Q1
lower_bound = Q1 - 1.5 * IQR
upper_bound = Q3 + 1.5 * IQR
outliers = df[(df['col'] < lower_bound) | (df['col'] > upper_bound)]
print(f"Anything outside ({lower_bound}, {upper_bound}) is an outlier")
```

### 1.2 value_counts() — 频率分布

```python
# 分类变量的值计数
df['category'].value_counts()
df['category'].value_counts(normalize=True)  # 百分比
df['category'].value_counts().plot(kind='bar')  # 柱状图
```

**用途：**
- 识别类别不平衡
- 发现稀有类别（Rare levels）
- 检查标签质量和拼写错误

### 1.3 均值 vs 中位数 — 何时用哪个

```python
import numpy as np

# 均值受异常值影响大
salary = [1, 4, 3, 2, 5, 4, 2, 3, 1, 500]
print(f"Mean: {np.mean(salary):.1f}")    # 52.5 — 被500拉高
print(f"Median: {np.median(salary):.1f}") # 2.5 — 反映真实水平
```

**决策规则：**
- 数据对称/近似正态 → 用均值
- 数据偏态/有异常值 → 用中位数
- 不确定时两个都报告

## 2. 相关性分析

### 2.1 协方差 (Covariance)

衡量两个变量共同变化的倾向。

```python
# 协方差矩阵
np.cov(df.radius_mean, df.area_mean)
# 或用 pandas
df.radius_mean.cov(df.area_mean)
```

- 协方差 > 0 → 正相关（同方向变化）
- 协方差 = 0 → 不相关（正交）
- 协方差 < 0 → 负相关（反方向变化）
- **缺点：** 有量纲，不同变量对之间不可比

### 2.2 Pearson 相关系数

协方差除以标准差之积，标准化到 [-1, 1]。

```python
# 方法一：直接调用
df[['area_mean', 'radius_mean']].corr(method='pearson')

# 方法二：手动计算验证
pearson = df.radius_mean.cov(df.area_mean) / (df.radius_mean.std() * df.area_mean.std())
```

**解读标准：**

| 相关系数绝对值 | 关系强度 |
|---------------|---------|
| 0.0 - 0.3 | 弱相关 |
| 0.3 - 0.7 | 中等相关 |
| 0.7 - 1.0 | 强相关 |

**适用条件：**
- 两个变量线性相关
- 变量近似正态分布
- 无显著异常值

### 2.3 Spearman 秩相关系数

基于秩（rank）而非原始值，更鲁棒。

```python
ranked_data = df.rank()
spearman_corr = ranked_data[['area_mean', 'radius_mean']].corr(method='pearson')
```

**何时用 Spearman 而非 Pearson：**
- 关系非线性（单调但非直线）
- 数据有异常值
- 数据是序数型（ordinal）
- Spearman 通常略微高于 Pearson（非线性时差异更明显）

### 2.4 相关性热力图

```python
import seaborn as sns
import matplotlib.pyplot as plt

plt.figure(figsize=(18, 18))
sns.heatmap(df.corr(), annot=True, linewidths=0.5, fmt=".1f", cmap='coolwarm')
plt.title('Correlation Map')
plt.xticks(rotation=90)
plt.yticks(rotation=0)
plt.show()
```

### 2.5 散点图矩阵 (Scatter Matrix / Pair Plot)

```python
from pandas.plotting import scatter_matrix

# 基础版
scatter_matrix(df[['radius_mean', 'area_mean', 'texture_mean']],
               figsize=(12, 8), diagonal='kde')

# 增强版 — seaborn PairGrid
sns.set(style="white")
g = sns.PairGrid(df[['radius_mean', 'area_mean', 'fractal_dimension_se']],
                  diag_sharey=False)
g.map_lower(sns.kdeplot, cmap="Blues_d")
g.map_upper(plt.scatter)
g.map_diag(sns.kdeplot, lw=3)
plt.show()
```

## 3. 假设检验框架

### 3.1 核心概念

**三步走：**

1. **量化效应大小** — 选择检验统计量（通常为均值差）
2. **定义零假设 (H₀)** — 假设效应不真实存在。如 "两组均值无差异" 或 "两变量独立"
3. **计算 p 值** — 在 H₀ 为真的假设下，观察到当前效应（或更极端）的概率

```
p 值 ≤ α (通常 0.05) → 拒绝 H₀，结果统计显著
p 值 > α              → 不能拒绝 H₀，证据不足
```

**零假设 (H₀)：** 想要推翻的假设。通常是"无差异"、"无关联"。
**备择假设 (H₁)：** 想要证明的假设。通常是"有差异"、"有关联"。

> "世界是平的"是 H₀。科学家们通过收集证据推翻 H₀，从而接受 H₁ "世界是圆的"。

### 3.2 p 值解读要点

⚠️ p 值 **不是**：
- 效应为真的概率
- H₀ 为真的概率
- 效应大小的度量

✅ p 值 **是**：
- 假设 H₀ 为真，观察到该效应的概率
- p < 0.05 意味着 "如果 H₀ 为真，我们观察到这种情况的概率不到 5%"
- 因此我们推断 H₀ 不太可能为真 → 拒绝 H₀

## 4. t 检验 — 比较两组均值

### 4.1 配对 t 检验 (Paired t-test)

同一组对象的前后对比或配对设计。

```python
from scipy import stats

statistic, p_value = stats.ttest_rel(group_before, group_after)
print(f't-statistic: {statistic:.4f}, p-value: {p_value:.4e}')

# p ≈ 0 → 拒绝 H₀，两组有显著差异
```

**适用场景：**
- 治疗前后对比
- 同一批样本在两个条件下的测量
- 配对设计实验

### 4.2 独立样本 t 检验 (Independent t-test)

两组独立样本比较。

```python
# 两组方差相等假设
statistic, p_value = stats.ttest_ind(group_a, group_b)

# Welch's t-test (不假设方差相等，更安全)
statistic, p_value = stats.ttest_ind(group_a, group_b, equal_var=False)
```

**适用场景：**
- 实验组 vs 对照组
- 男 vs 女、患病 vs 健康
- 两组独立样本的任何比较

**前提假设：**
- 两组近似正态分布（大样本可放宽）
- 独立样本（Welch 版本不要求方差齐性）

## 5. ANOVA — 多组均值比较

当需要比较三组及以上时，不能重复使用 t 检验（多重比较会膨胀 I 类错误）。

```python
from scipy import stats

# 单因素 ANOVA
f_statistic, p_value = stats.f_oneway(group_a, group_b, group_c)
print(f'F-statistic: {f_statistic:.4f}, p-value: {p_value:.4f}')
```

**适用场景：**
- 比较 3+ 个治疗方案的效果
- 比较多个地区的销售差异
- 比较多个分类下的连续变量

**事后检验 (Post-hoc)：**
```python
# ANOVA 显著后，确定哪两组有差异
from statsmodels.stats.multicomp import pairwise_tukeyhsd

tukey = pairwise_tukeyhsd(
    endog=df['value'],       # 因变量
    groups=df['group'],      # 分组变量
    alpha=0.05
)
print(tukey)
```

## 6. 卡方检验 (Chi-Square Test) — 分类变量关联

检验两个分类变量是否独立。

```python
from scipy.stats import chi2_contingency

# 构造列联表
contingency_table = pd.crosstab(df['category_a'], df['category_b'])

# 卡方检验
chi2, p_value, dof, expected = chi2_contingency(contingency_table)
print(f'Chi2: {chi2:.4f}, p-value: {p_value:.4f}, df: {dof}')
```

**适用场景：**
- 性别与购买行为是否相关
- 治疗方案与治愈率是否相关
- 两分类变量是否独立

**前提：**
- 期望频数 < 5 的单元格不超过 20%
- 每个单元格期望频数 ≥ 1

## 7. 效应量 (Effect Size) — 差异有多大

p 值告诉你 "是否有差异"，效应量告诉你 "差异有多大"。两者互补，缺一不可。

### 7.1 Cohen's d — 两组均值差异的效应量

```python
import numpy as np

mean_diff = group_malignant.mean() - group_benign.mean()
var_pooled = (len(group_benign) * group_benign.var() +
              len(group_malignant) * group_malignant.var()) / \
             (len(group_benign) + len(group_malignant))

cohens_d = abs(mean_diff) / np.sqrt(var_pooled)
print(f"Cohen's d: {cohens_d:.4f}")
```

**Cohen 建议的解释标准：**

| Cohen's d | 效应大小 | 含义 |
|-----------|---------|------|
| 0.2 | 小 (Small) | 差异微弱 |
| 0.5 | 中 (Medium) | 差异可见 |
| 0.8 | 大 (Large) | 差异显著 |
| ≥ 2.0 | 极大 | 两组几乎完全不同群体 |

### 7.2 何时报告效应量

- 总是与 p 值一起报告
- p 显著但 d 很小 → 样本量大导致，实际意义有限
- p 不显著但 d 较大 → 可能样本量不足，值得进一步研究
- 专业报告标配：`t(98) = 3.45, p = 0.001, d = 0.69`

## 8. 正态性检验 — 选择检验方法的前提

许多参数检验假设数据服从正态分布。正态性检验帮助判断该假设是否成立。

### 8.1 Shapiro-Wilk 检验

最常用的正态性检验，适用于小到中等样本（n < 5000）。

```python
from scipy import stats

statistic, p_value = stats.shapiro(data)
print(f'Shapiro-Wilk: W={statistic:.4f}, p={p_value:.4f}')

if p_value > 0.05:
    print("不能拒绝正态性假设 → 数据可能正态")
else:
    print("拒绝正态性假设 → 数据非正态，考虑非参数检验")
```

### 8.2 其他正态性检验

```python
# D'Agostino's K² 检验 (样本量较大时)
from scipy.stats import normaltest
statistic, p_value = normaltest(data)

# Kolmogorov-Smirnov 检验 (与理论分布比较)
from scipy.stats import kstest
statistic, p_value = kstest(data, 'norm', args=(data.mean(), data.std()))
```

### 8.3 图形化判断

```python
import matplotlib.pyplot as plt
from scipy import stats

# Q-Q 图
fig, (ax1, ax2) = plt.subplots(1, 2, figsize=(12, 5))
stats.probplot(data, dist="norm", plot=ax1)
ax1.set_title('Q-Q Plot')

# 直方图 + 正态拟合
ax2.hist(data, bins=30, density=True, alpha=0.6, color='g')
mu, sigma = stats.norm.fit(data)
x = np.linspace(data.min(), data.max(), 100)
ax2.plot(x, stats.norm.pdf(x, mu, sigma), 'r-', lw=2)
ax2.set_title('Histogram with Normal Fit')
plt.show()
```

## 9. 正态分布与 z-score

### 9.1 正态分布特征
- 钟形曲线，对称
- 均值 = 中位数 = 众数
- 68% 数据在 ±1σ 内，95% 在 ±2σ 内，99.7% 在 ±3σ 内

### 9.2 z-score 计算与查表

```python
# z = (x - μ) / σ
from scipy import stats

z_score = (value - np.mean(data)) / np.std(data)
percentile = stats.norm.cdf(z_score) * 100

# 例：IQ测试 μ=110, σ=20
# 80到140之间的比例
z1 = (80 - 110) / 20  # -1.5
z2 = (140 - 110) / 20  # 1.5
proportion = stats.norm.cdf(z2) - stats.norm.cdf(z1)
print(f"{proportion*100:.1f}% 的人在 80-140 之间")  # 86.6%
```

## 10. 检验方法决策框架 — When to Use Which Test

```
需要分析什么？
│
├── 描述数据分布
│   ├── 数值型 → describe(), 直方图, 箱线图
│   └── 分类型 → value_counts(), 柱状图
│
├── 比较组间差异（连续因变量）
│   ├── 2组
│   │   ├── 独立样本 → Independent t-test (ttest_ind)
│   │   └── 配对样本 → Paired t-test (ttest_rel)
│   ├── 3+组
│   │   ├── 正态+方差齐 → One-way ANOVA (f_oneway)
│   │   └── 非正态 → Kruskal-Wallis H-test
│   └── 多因素 → Two-way ANOVA / Factorial ANOVA
│
├── 检验分类变量关联
│   ├── 2×2 列联表 → Chi-square test (chi2_contingency)
│   └── 小样本期望频数不足 → Fisher's Exact Test
│
├── 分析变量间关系（两连续变量）
│   ├── 线性+正态 → Pearson correlation
│   ├── 非线性/有异常值 → Spearman rank correlation
│   └── 可视化 → 散点图 / PairGrid
│
├── 检验数据正态性
│   ├── n < 5000 → Shapiro-Wilk
│   ├── n ≥ 5000 → D'Agostino K²
│   └── 图形辅助 → Q-Q Plot / 直方图
│
└── 量化差异大小
    ├── 两组均值 → Cohen's d
    ├── 多组 → η² (Eta-squared)
    └── 分类关联 → Cramér's V (基于卡方)
```

### 快速速查表

| 检验方法 | 函数 | 因变量 | 自变量 | 前提 |
|---------|------|--------|--------|------|
| 配对 t 检验 | `ttest_rel` | 连续 | 2组(配对) | 差值近似正态 |
| 独立 t 检验 | `ttest_ind` | 连续 | 2组(独立) | 正态, 方差齐(可选) |
| Welch t 检验 | `ttest_ind(equal_var=False)` | 连续 | 2组(独立) | 近似正态 |
| One-way ANOVA | `f_oneway` | 连续 | 3+组 | 正态, 方差齐 |
| Kruskal-Wallis | `kruskal` | 连续/序数 | 3+组 | 无(非参数) |
| 卡方检验 | `chi2_contingency` | 分类 | 分类 | 期望频数 ≥ 5 |
| Pearson r | `corr(method='pearson')` | 连续 | 连续 | 线性, 正态 |
| Spearman ρ | `corr(method='spearman')` | 连续/序数 | 连续/序数 | 单调关系 |
| Shapiro-Wilk | `shapiro` | — | — | n < 5000 |

## 11. 完整分析示例 — 端到端统计报告

```python
import pandas as pd
import numpy as np
from scipy import stats
import matplotlib.pyplot as plt
import seaborn as sns

# ===== 0. 数据加载与描述 =====
df = pd.read_csv("data.csv")
print("=== 描述性统计 ===")
print(df.describe())
print("\n=== 分类变量分布 ===")
print(df['diagnosis'].value_counts())

# ===== 1. 正态性检验 =====
stat, p_norm = stats.shapiro(df['value'])
print(f"\nShapiro-Wilk: W={stat:.4f}, p={p_norm:.4f}")
is_normal = p_norm > 0.05
print(f"数据{'符合' if is_normal else '不符合'}正态分布")

# ===== 2. 选择检验方法 =====
group_b = df[df['diagnosis'] == 'B']['value']
group_m = df[df['diagnosis'] == 'M']['value']

if is_normal:
    # 参数检验：t-test
    t_stat, p_val = stats.ttest_ind(group_b, group_m)
    test_name = "Independent t-test"
else:
    # 非参数检验：Mann-Whitney U
    t_stat, p_val = stats.mannwhitneyu(group_b, group_m, alternative='two-sided')
    test_name = "Mann-Whitney U"

# ===== 3. 效应量 =====
mean_diff = group_m.mean() - group_b.mean()
var_pooled = (len(group_b) * group_b.var() + len(group_m) * group_m.var()) / \
             (len(group_b) + len(group_m))
cohens_d = abs(mean_diff) / np.sqrt(var_pooled)

# ===== 4. 报告 =====
print(f"\n=== 统计检验报告 ===")
print(f"检验方法: {test_name}")
print(f"检验统计量: {t_stat:.4f}")
print(f"p-value: {p_val:.4e}")
print(f"Cohen's d: {cohens_d:.4f}")
print(f"显著性: {'显著 ★' if p_val < 0.05 else '不显著'}")
print(f"效应大小: {'大' if cohens_d >= 0.8 else '中' if cohens_d >= 0.5 else '小'}")
```

## 12. 常见陷阱与检查清单

| 陷阱 | 症状 | 修复 |
|------|------|------|
| 多重比较不做校正 | 假阳性率膨胀 | ANOVA 后用 Tukey HSD；多次 t 检验用 Bonferroni 校正 |
| 非正态数据用 t 检验 | 结论不可靠 | 先做正态性检验，非正态用 Mann-Whitney 或 Kruskal-Wallis |
| 只看 p 不看效应量 | 大样本几乎总是显著 | 始终报告 Cohen's d / η² / Cramér's V |
| 将 p > 0.05 解释为 "无差异" | 忽略检验功效不足 | 报告"证据不足以拒绝 H₀"，而非"无差异" |
| 相关 ≠ 因果 | 误导性结论 | 仅描述关联强度，避免因果语言 |
| 配对数据用独立检验 | 检验功效下降 | 配对设计用 `ttest_rel` |
| 期望频数不足仍用卡方 | p 值不准确 | 用 Fisher's Exact Test |
| 混淆 Pearson 和 Spearman | 非线性关系被低估 | 先画散点图判断线性，再选方法 |

## Quality Checklist
- [ ] 是否先做了描述性统计（describe + value_counts）？
- [ ] 是否检验了正态性（Shapiro-Wilk / Q-Q plot）？
- [ ] 是否根据数据类型和分布选择了正确的检验方法？
- [ ] 是否同时报告了 p 值和效应量？
- [ ] 是否进行了多重比较校正（如果做了多次检验）？
- [ ] 相关系数是否区分了 Pearson 和 Spearman 的使用场景？
- [ ] p > 0.05 时是否避免声称"无差异"？
- [ ] 是否检查了检验的前提假设？
