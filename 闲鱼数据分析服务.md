## Skill Content

# Xianyu Data Analysis Service Test Framework

## Applicable Scenarios
- Data analysis professionals wanting to monetize skills
- Testing market acceptance of service products
- Low-cost, quick-start business model validation
- Acquiring first batch of customers on Xianyu platform

## Core Principles
1. **Low-risk trial**: Zero capital investment, controllable time
2. **Quick validation**: See market response within 1 week
3. **Gradient design**: Traffic product → Main product → Profit product
4. **Data-driven**: Daily tracking, weekly adjustments

## Four-Step Execution Framework

### Step 1: Market Analysis & Product Design (Day 1)

#### 1.1 Xianyu Platform Characteristics
**Advantages**:
- Zero-cost startup, no deposit required
- Precise traffic, "data analysis" has stable demand
- Flexible transactions, multiple payment methods
- Low trial cost, quick adjustments possible

**Challenges**:
- Price sensitive users accustomed to low prices
- Difficult to build trust, need case accumulation
- Chaotic competition with low-price, low-quality services

#### 1.2 Three-Gradient Product Design
**Product 1: Traffic Product (9.9 RMB)**
- Name: Quick Data Diagnosis
- Content: Automated AI diagnostic — 3 core data problems identified, 1 prioritized optimization suggestion, delivered as a 6-page PDF report with embedded charts
- Delivery: PDF report only, no consultation/video included
- Goal: Acquire first batch of customers, accumulate reviews

**Product 2: Main Product (99 RMB)**
- Name: Professional Data Analysis Report
- Content: 10-15 page PDF report + 5-8 professional charts + PPT presentation + Excel data analysis table, actionable recommendations
- Delivery: PDF report + PPT + Excel files. No consultation/video included.
- Goal: Main revenue source

**Product 3: Profit Product (199 RMB)**
- Name: Custom Deep Analysis
- Content: 20-30 page deep analysis PDF + PPT + data model files, trend prediction, scenario simulation, ROI calculation
- Delivery: PDF report + PPT + data files. No consultation/video included.
- Goal: High-value customers, brand building

**Service Delivery Policy (CRITICAL)**:
- Deliverables are FILES ONLY: PDF reports, PPT presentations, Excel data tables
- NO video walkthroughs, NO online consultations, NO phone calls
- All communication via Xianyu chat (text only)
- Turnaround: 9.9 RMB = instant (~5 min AI), 99 RMB = 3-5 days, 199 RMB = 5-7 days

#### 1.3 Product Copywriting Essentials
- **Title**: Include 【】symbols, highlight core value
- **Description**: Pain point scenarios → Solutions → Service content → Success cases → Call to action
- **Trust elements**: Professional background, service cases, guarantee commitments
- **Conversion guidance**: Clear purchase steps and contact information

### Step 2: Visual & Operational Preparation (Day 1)

#### 2.1 Image Production Guide
**9 Standard Images**:
1. Cover image (attract clicks)
2. Pain point scenario image (create resonance)
3. Service content image (clear display)
4. Deliverable showcase image (report samples)
5. Service flow chart image (transparent process)
6. Success case image (results proof)

**Design Specifications**:
- Size: 750x1000 pixels
- Color scheme: Blue-white tech style (main color blue, secondary white)
- Fonts: Source Han Sans Bold (title), Regular (body)
- Icons: Flaticon or Canva built-in icons

#### 2.2 Pricing Strategy
**Pricing Principles**:
- More professional than low-price competitors, cheaper than professional competitors
- Use psychological pricing (9.9, 99, 199)
- Set limited-time offers and package combinations

**Promotion Strategy**:
- Cold start period: New customer discounts (special price for first 10)
- Growth period: Limited-time discounts (10% off every Tuesday)
- Existing customer maintenance: Repurchase discounts, referral rewards

#### 2.3 Account Optimization
- **Nickname**: Data Analyst/Data Analysis Assistant
- **Bio**: Highlight professional background and service areas
- **Avatar**: Professional headshot or data analysis related icon
- **Credit**: Complete Alipay credit information

### Step 3: Execution & Optimization (Week 1-2)

#### 3.1 Daily Operations
**Day 0**:
- Complete personal profile
- Create product images (Canva)
- List first product (9.9 RMB diagnosis)

**Day 1**:
- List other two products
- Optimize titles and tags
- Post promotions in relevant "fish ponds"

**Day 2-7**:
- 30 minutes daily Xianyu maintenance
- Quick response to all inquiries (within 5 minutes)
- Actively search for needs and contact potential customers
- Proactively request reviews after service completion

#### 3.2 Key Success Factors
1. **Quick response**: Xianyu users expect immediate replies
2. **Trust building**: Timely delivery, proactive review requests, professional display
3. **Data tracking**: Record key metrics daily, analyze and adjust weekly

#### 3.3 Common Issue Responses
**Price objections**:
- "Too expensive" → Value emphasis: "Compared to opportunity cost, this investment is worthwhile"
- "Others are cheaper" → Differentiation: "We provide professional analysis, not simple data organization"
- "I'll think about it" → Urgency: "This is a limited-time offer, price returns to normal tomorrow"

**Service questions**:
- Clearly define service boundaries and delivery standards
- Set reasonable delivery time (3-5 days)
- Provide one free revision opportunity

### Step 4: Data Tracking & Iteration (Ongoing)

#### 4.1 Key Metrics Tracking
**Daily tracking**:
- Page views, inquiries, transactions
- Inquiry response rate, transaction conversion rate
- Average order value, repurchase rate

**Weekly analysis**:
- Efficiency of each conversion funnel stage
- Product popularity ranking
- Customer feedback and demand insights

#### 4.2 Data Tracking Table Design
Includes:
- Daily operation data records
- Customer information registration table
- Financial income details
- Problem and optimization records
- Key learnings and insights

#### 4.3 Iteration Optimization Mechanism
**Conditions triggering price adjustments**:
1. Supply shortage (inquiries >5/day, conversion rate >30%)
2. Competition changes (many imitators appear)
3. Cost increase (service time extends)
4. Brand establishment (accumulate 50+ positive reviews)

**Optimization directions**:
1. Optimize service processes based on customer feedback
2. Adjust pricing strategy based on data performance
3. Expand product line to meet细分需求
4. Increase promotion channels for more traffic

## Expected Results & Risk Control

### First Month Goals (Conservative)
- Total revenue: 2000-3000 RMB
- Transactions: 15-20 orders
- Positive reviews: 5-10
- Net profit margin: 60-70%

### Risk Control Measures
1. **Time risk**: Control daily investment to 30-60 minutes
2. **Capital risk**: Zero capital investment, only time cost
3. **Market risk**: 1-week quick validation, timely adjustments
4. **Reputation risk**: Refund if dissatisfied, proactive communication

### Failure Contingency Plans
- **No inquiries**: Adjust titles and images, actively search for needs
- **No transactions**: Lower prices or add service content
- **Negative reviews**: Immediately improve service, proactively communicate solutions

## 9.9 Yuan Diagnostic Service Workflow

When a customer orders the 9.9 yuan service, follow this exact workflow:

### Step 1: Data Snapshot (1 min)
- Print column names, first 5 rows, shape
- Auto-detect data types (numeric/categorical/temporal)

### Step 2: Three Core Findings (3-5 min)
Run analysis focused on three dimensions:
1. **Sales/Orders**: anomalies, peaks, trends (7-day MA)
2. **Customer**: repurchase rate, concentration (top 10% share)
3. **Operations**: delivery delays, bottleneck identification

Each finding gets one chart, saved as PNG.

### Step 3: One Prioritized Recommendation (2 min)
- Must be specific and actionable (not generic "optimize conversion rate")
- Include estimated impact if possible
- End with clear upsell to 99 yuan full report

### Output Format
- 1 markdown report (3-5 pages) + 3-4 charts (PNG)
- **PDF delivery**: Use `templates/diagnostic_pdf.py` to convert to a professional 6-page PDF with embedded charts, cover page, and upsell CTA. See `references/pdf-generation-guide.md` for fpdf2 installation (including proxy workaround), design specs, and pitfalls.
- Report structure: Data Overview → Finding 1 → Finding 2 → Finding 3 → Recommendation → Upgrade CTA
- Target turnaround: under 5 minutes end-to-end
