## Skill Content

# 迁移学习 (Transfer Learning with Keras)

## Purpose
在小数据集上利用 ImageNet 预训练模型实现高精度图像分类。两种策略：**Feature Extraction**（冻结卷积基）和 **Fine-Tuning**（解冻顶层）。

## When to use
- 图像分类且训练数据 < 5000 张
- 不想从零训练 CNN
- 需要快速达到高精度
- 用户说"图像分类"、"数据太少"

## Core Workflow

### Step 1: 选择预训练模型
```python
from tensorflow.keras.applications import (
    VGG16, ResNet50, EfficientNetB0, MobileNetV2, InceptionV3
)

# 选择指南
# 精度优先: EfficientNetB0 > ResNet50 > VGG16
# 速度优先: MobileNetV2 (移动端)
# 平衡: ResNet50
```

### Step 2: Feature Extraction (冻结卷积基)
```python
from tensorflow.keras import layers, models, optimizers

# 加载预训练模型（不含分类头）
IMG_SIZE = 224
base_model = ResNet50(
    weights='imagenet',
    include_top=False,           # 去掉原始分类头
    input_shape=(IMG_SIZE, IMG_SIZE, 3)
)
base_model.trainable = False     # 冻结！不训练

# 构建新分类头
model = models.Sequential([
    base_model,
    layers.GlobalAveragePooling2D(),
    layers.Dense(256, activation='relu'),
    layers.Dropout(0.5),
    layers.Dense(num_classes, activation='softmax'),
])

model.compile(
    optimizer=optimizers.Adam(learning_rate=0.001),
    loss='categorical_crossentropy',
    metrics=['accuracy']
)

# Quick train (only classification head)
history = model.fit(train_generator, epochs=10, validation_data=val_generator)
```

### Step 3: 数据预处理
```python
from tensorflow.keras.preprocessing.image import ImageDataGenerator

# 关键：必须用与预训练模型相同的预处理
from tensorflow.keras.applications.resnet50 import preprocess_input

# ImageNet 预处理：缩放 + 减均值
train_datagen = ImageDataGenerator(
    preprocessing_function=preprocess_input,
    rotation_range=20,
    width_shift_range=0.2,
    height_shift_range=0.2,
    zoom_range=0.2,
    horizontal_flip=True,
    validation_split=0.2,
)

train_gen = train_datagen.flow_from_directory(
    'data/train',
    target_size=(IMG_SIZE, IMG_SIZE),
    batch_size=32,
    class_mode='categorical',
    subset='training',
)
```

### Step 4: Fine-Tuning (解冻顶层)
```python
# Feature extraction 训练收敛后 → 微调
base_model.trainable = True

# 只解冻最后几层（前层是通用特征，不需要调）
fine_tune_at = 100  # 解冻最后100层
for layer in base_model.layers[:fine_tune_at]:
    layer.trainable = False

# 低学习率！防止破坏预训练权重
model.compile(
    optimizer=optimizers.Adam(learning_rate=1e-5),  # 比之前低100倍！
    loss='categorical_crossentropy',
    metrics=['accuracy']
)

# 继续训练
history_fine = model.fit(
    train_gen, epochs=10,
    validation_data=val_gen,
    initial_epoch=history.epoch[-1]  # 续接之前训练
)
```

### Step 5: 多模型对比
```python
def build_transfer_model(base_model_class, name):
    base = base_model_class(weights='imagenet', include_top=False,
                            input_shape=(IMG_SIZE, IMG_SIZE, 3))
    base.trainable = False
    model = models.Sequential([
        base, layers.GlobalAveragePooling2D(),
        layers.Dense(256, activation='relu'),
        layers.Dropout(0.5),
        layers.Dense(num_classes, activation='softmax'),
    ])
    model.compile(optimizer='adam', loss='categorical_crossentropy', metrics=['accuracy'])
    return name, model

models_dict = dict([
    build_transfer_model(ResNet50, 'ResNet50'),
    build_transfer_model(EfficientNetB0, 'EfficientNetB0'),
    build_transfer_model(MobileNetV2, 'MobileNetV2'),
])

for name, model in models_dict.items():
    model.fit(train_gen, epochs=5, validation_data=val_gen, verbose=0)
    acc = model.evaluate(val_gen, verbose=0)[1]
    print(f"{name}: {acc:.3f}")
```

## Decision Framework

| 场景 | 策略 | 学习率 |
|------|------|--------|
| 数据 < 500，与 ImageNet 相似 | Feature Extraction | 0.001 |
| 数据 500-2000，与 ImageNet 相似 | Feature Extraction + Fine-Tune | 0.001 → 1e-5 |
| 数据 > 2000 | Fine-Tune from start | 1e-4 |
| 数据与 ImageNet 差异大 | Fine-Tune more layers | 1e-4 |

## Common Pitfalls
1. 不匹配预处理 → 用 preprocess_input
2. 解冻后学习率太高 → 必须降到 1e-5
3. 冻结时忘记 compile → 重新 compile 才生效
4. 小 batch_size 用大模型 → OOM，用梯度累积或减小 batch
