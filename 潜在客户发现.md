## Skill Content

# Lead Research Assistant

帮你发现数据分析和自动化服务的高质量潜在客户。

## When to Use

- 想在 Upwork/闲鱼/Fiverr 上找客户
- 需要评估某个行业对数据分析的需求
- 准备做 Outreach（主动联系潜在客户）
- 分析竞品服务定价和策略

## What This Skill Does

1. **分析你的服务**：理解你的数据分析/自动化能力栈
2. **识别目标平台**：Upwork、闲鱼、猪八戒、和鲸等
3. **搜索匹配项目**：关键词搜索 → 需求匹配 → 竞争分析
4. **生成提案策略**：个性化话术、定价建议、差异化优势
5. **追踪线索**：记录和跟进潜在客户

## How to Use (via Hermes terminal)

### Basic: Search Upwork for data analysis jobs
```
user: "帮我在 Upwork 上找 5 个数据分析项目"
→ Hermes: browser_harness → upwork.com → search "data analysis" → filter → list top 5
```

### Advanced: Targeted lead search
```
user: "找供应链/快消行业需要库存分析的公司"
→ Hermes: web search → identify companies → check job boards → generate list
```

### Competitive Analysis
```
user: "分析闲鱼上数据分析服务的定价和评论"
→ Hermes: browser_harness → 闲鱼 → search → collect pricing, reviews → generate report
```

## Instructions for Agent

1. 询问用户的服务定位（擅长什么，什么行业）
2. 确定目标平台（国内 vs 国际）
3. 搜索匹配的需求/项目
4. 分析竞争环境和定价
5. 生成优先级排序的线索列表
6. 为每个线索生成个性化提案草稿
